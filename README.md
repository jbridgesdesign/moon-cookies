## 📖 About

Moon Cookies is a fictional cookie brand homepage designed to practice **responsive web design** and core **HTML & CSS** skills, while also serving as a portfolio piece. The site is vibrant and playful, featuring AI-generated cookie imagery, a promotional banner, a features section, and a flavor showcase.

---

## ✨ Features

- **Responsive Layout** — Separate desktop and mobile navigation for a seamless experience across screen sizes
- **Hero Banner** — Promotional spotlight for the TikTok-famous "Dark Crystal" cookie with a CTA button
- **Features Section** — Highlights the brand's fresh ingredients and colorful, creative cookie offerings
- **Flavor Showcase** — A visual gallery displaying 4 of 10+ cookie flavors
- **Testimonial Section** — A fun social-proof quote from a "Famous TikToker"
- **Custom Typography** — Google Fonts integration (*Damion*, *Rubik*, and *Roboto*)
- **Favicon** — Custom moon icon for a polished browser tab experience

---

## 🛠️ Built With

- HTML5 (semantic elements: `header`, `main`, `footer`, `nav`)
- CSS3 (custom stylesheet with responsive breakpoints)
- Google Fonts
- AI-generated images via **Google Gemini's Nano Banana 2**
